# ⚡ Prep Pulse V26 — NEET TRACKER

**All-in-one NEET preparation tracker — works offline, installable as PWA.**

## 🚀 Use Directly
Open `PrepPulse_V26.html` in Chrome/Safari — no server needed.

## 📱 Install as App (PWA)

### Android (Chrome)
1. Open the app in Chrome  
2. Tap ⋮ menu → **"Add to Home Screen"**

### iPhone (Safari)
1. Open in Safari  
2. Tap Share → **"Add to Home Screen"**

### Windows/Mac (Chrome)
1. Open in Chrome  
2. Click `+` icon in address bar → Install

## 🌐 Host on GitHub Pages (Recommended)
1. Upload all files to a GitHub repo  
2. Go to **Settings → Pages → Source: main branch**  
3. Your app URL: `https://yourusername.github.io/repo-name/PrepPulse_V26.html`

## 📁 Files
| File | Purpose |
